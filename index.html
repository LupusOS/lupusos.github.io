Welcome to LupusOS

LupusOS is a lightweight, community-driven Linux distribution powered by the PISI package management system. Inspired by Pardus 2011, it combines modern AI readiness with a futuristic, user-friendly design for developers and enthusiasts.

Motivation Behind LupusOS

Why create LupusOS in a crowded Linux landscape?

  1. Pardus 2011 offered a lightweight, user-friendly vision that LupusOS modernizes for 2025.
  2. Many distributions suffer from bloat or complexity; LupusOS prioritizes simplicity and performance.
  3. The PISI package manager offers a unique and efficient solution for seamless software management.
  4. AI readiness meets the needs of developers and data scientists with pre-configured tools.

LupusOS Features

  • Lightweight live ISO under 2GB, optimized for modern and legacy hardware.
  • PISI package management for seamless software installation and updates.
  • AI-ready with pre-configured tools for machine learning and data science.
  • Beautiful, customizable desktop environments (e.g., LXQt) with out-of-the-box usability.
  • Community-driven development with transparent contributions on GitHub.
  • Robust security with verified source checksums and CVE monitoring.
  • Futuristic design inspired by Pardus 2011, evolved for modern users.

How Is LupusOS Different?

  • Unlike Pardus, LupusOS leverages the lightweight PISI package manager for streamlined software management.
  • AI-ready out of the box, with tools for developers and data scientists.
  • Live ISO with offline installation, unlike net-installer-heavy distributions.
  • Community-driven, with open development on GitHub.
  • Minimalist design avoids bloat while maintaining modern functionality.
